4C™ – Deployment

Skywind 4C™ is a cloud-based solution built with the most advanced SAP technologies, such as SAPUI5, FIORI, ODATA, ABAP.

Some key technical data is provided below.

Security:

  • Secured (https://) two-factor authentication – access from anywhere
  • Secured VPN (IPSec) tunnel between Skywind 4C™ and client’s SAP routers
  • Encrypted (SNC) RFC connection to client’s SAP systems
  • Advanced  Authentication and Authorization mechanisms
  • Zero footprint – ALL processes data can be stored on client’s SAP instances
  • GDPR compliant – EVERY field in EACH alert can be masked

Mobility:

  • Running on Amazon Web Services
  • Unlimited hardware resources
  • Mobile-friendly interface
  • Close-to-zero footprint on your SAP systems
  • Adaptive SAPUI5 interface for all devices
