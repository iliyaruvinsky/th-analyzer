Alert management
Alert management
popover contains several panels providing information and analytical capabilities for an Alert chosen in
Cockpit:
Metadata
Scheduling
Email
SMS
Summary
History
About
Metadata
Metadata
panel displays the overall information about the chosen Alert and its parameters. For more information about Alert parameters please refer to
Wizard – Parameters
section.
From the bottom frame you can adjust them using
Wizard
and
Export “Metadata” to Excel
using the respective buttons:
Scheduling
Once you click on a value in the
Scheduling
column in
Cockpit
the
Alert management Scheduling
panel will be displayed.  It contains information about Regular and Exception Scheduling. For more information please refer to
Wizard – Scheduling
section.
From the bottom frame you can adjust Scheduling using
Wizard
and
Export “Scheduling” to Excel
using the respective buttons:
Email
Once you click on a value in the
Email
column in
Cockpit
the
Alert management Email
panel will be displayed. It consists of 3 sections to display Recipients, Dynamic recipient list (DRL) and Content-dependent distribution filter CDD. For more information about them please refer to
Wizard – Recipients
section. From the bottom frame you can adjust the recipients using
Wizard
and
Export “Email” to Excel
using the respective buttons:
SMS
Once you click on a value in the
SMS
column in
Cockpit
the
Alert management SMS
panel will be displayed. For more information about managing Recipients, Recipient lists, Content-dependant distribution and Dynamic Recipient Lists please refer to
Wizard – Recipients
section.
From the bottom frame you can adjust the recipients using
Wizard
and
Export “SMS” to Excel
using the respective buttons:
Summary
Once you click on a value in the
Summary
column in
Cockpit
the
Alert management – Summary
panel will be displayed.
The buttons located at the upper right part of the frame are used to show/hide the sub-panels used for displaying and managing of the Alert’s output:
Metadata
for displaying Alert’s Metadata.
Parameters
for displaying Alert’s and Exception Indicator’s parameters.
Format Output
for managing the fields of the Alert’s output.
Data
for displaying the output data values of the Alert. This sub-panel is displayed as default.
When you click on several buttons, several respective sub-panels will be displayed:
Using the buttons at the bottom panel you can:
Save format output
Wizard
– open Alert Wizard for changing the Alert. For more details please refer to the
relevant section
Export “Summary” to Excel
Metadata
sub-panel displays the following information about the Alert:
Parameters
show Alert and Exception Indicator Parameters. To read more about Alert Instances and Excpetion Indicators please refer to
4C™ – Glossary.
Format output
sub-panel helps you to manage the Alert output fields in the same manner as using
Wizard – Format
functionality:
Data
sub-panel displayes the complete output result of the Alert runs. To navigate the table, sort and manage filters you can use the respective buttons and table header:
Click the
<Field names>
button to display the techincal names of the fields at the table header:
You can display any of the report’s rows in a popover by clicking on the leftmost column icon  against the desired report row. It is convenient to look at all the data without scrolling and additionally you can export data to Excel:
History
History
panel provides the information about occured events, that were caught by Alerts in the past. History is being presented in a graphical form (Line Chart) with capabilities to drill down to the details.
Buttons located at the upper right part of the frame are used to show/hide the sub-panels used to display and manage Alert’s output:
Metadata
for displaying Alert’s Metadata.
Parameters
for displaying Alert’s and Exception Indicator’s parameters.
Data
for displaying the output data values of the Alert. This sub-panel is displayed as default.
When you click on several buttons, several respective sub-panels will be displayed:
Using the buttons at the bottom panel you can:
Wizard
– open Alert Wizard for changing the Alert. For more details please refer to the
relevant section
Export “Summary” to Excel
Metadata
sub-panel displays the following information about the Alert:
Parameters
sub-panel shows Alert and Exception Indicator Parameters. To read more about Alert Instances and Excpetion Indicators please refer to
4C™ – Glossary.
Data
sub-panel displays the chart with data points for each alert run (left frame) and numerical presentation (right frame):
From here you can drill down to detailed data by clicking on a data point at the chart and then on the
<Data>
button in the appeared popover or on the
<Data>
button against a chosen Alert run at the right frame:
The detailed data is displayed in the popopver:
The data can be displayed for any time range. Use the buttons above the chart to choose one of the default ranges or a custom one:
For any time range (except for
Today
) you can compare the Alert against other related ones. To do this, start with clicking on the
<Analyze related alerts>
button:
Then check the checkboxes against the desired Alerts and click
<Refresh lines>
button:
To select a custom date range click on the
<Custom>
button and then input the range by manual input or using the calendar selection:
