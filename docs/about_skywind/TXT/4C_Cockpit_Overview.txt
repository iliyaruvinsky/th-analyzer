Cockpit – Overview
Alerting Cockpit
(or just Cockpit) contains all operative information about alerts that run in your SAP system/s.
You can analyze various aspects of alerts like:
Metadata
Scheduling
Email Recipients
SMS Recipients
Alert Events Summary
Alert Events History
About (Alert’s Documentation)
The most important data about Alert Instances is available immediately with an option to drill down to the details and perform further analysis.
The upper part of the frame contains functions for filtering and managing the presentation of Alerts in the Cockpit.
Click on the
<Filters>
button to select alerting period and filter on specific alerts recipients. You can change the alerting period range by manual input or by selecting the dates range via Calendar selector:
Use the
<Recipients>
filter to focus on specific recipients:
Use
<Reset>
button to reset the filters:
You can switch between
Table
and
Hierarchy
display of Alerts by clicking on the respective button. All alerts are organized in an hierarchy, under Category >> Sub Category, so when using the
Hierarchy
display type, you will have to extend the specific leaves of the hierarchy tree.
Use the
<Arrow>
buttons to navigate to the first/last row/column of the table:
Use the following dropdown list to choose Alerts by Categories and Sub-Categories:
Manage and remove filters and sorting using the following buttons:
The table containing the Alert information occupies the central part of the frame. To find specific alert by its description, use the header of the table, (as anywhere across the system):
Source system
column provides you with information about the mointored SAP systems. Each alert runs against a single Source system:
Summary
column shows the total number of collected events for all runs of an Alert during the selected date/time period. By default, Alerting Cockpit is set to present daily occurrences (what happened today), according to your local browser time. The date/time range can be changed by using the respective filter. Filters usage is described in the above section of this thread.
Click on a value to drill down to the detailed information at the respective panel of the
Alert management
window. For more information please refer to
Alert management – Summary
Last run
column shows the number of events, collected during the last run of the Alert only:
Activity
column shows the Alerts’ statuses (whether they are Active or Inactive). By default, only active alerts are displayed:
Severity
column shows the Alerts’ severity settings. The severity can be changed by authorized user/s in Alert Wizard.
Scheduling
column displays the scheduling frequency of the Alerts. Click on a value to open the detailed information at the respective panel of the
Alert management
window. For more information please refer to
Alert management
section:
Mode
column displays whether an Alert runs in Full or Delta mode. Alerting modes (Full / Delta) are described in the
Wizard – Basic Settings
section of the documentation:
Email
column shows if an Alert’s notification output is scheduled to be sent by Email and the number of Email recipients, subscribed to an alert. Click on the “Email” icon to open the detailed information in the respective panel of the
Alert management
window. For more information please refer to
Alert management
section:
SMS
column shows if  information, placed in Alert’s header is scheduled to be sent as text message and the number of SMS recipients, subscribed to receive this info. Click on the “SMS” icon to open the detailed information in the respective panel of the
Alert management
window. For more information please refer to
Alert management
section:
Specifications
section incorporates 2 columns: one to open
Metadata
panel in the
Alert management
window while the other opens
Edit
context menu:
Favorites
column allows to mark specific Alerts as Favorites. After marking specific alerts, you can click on the <
Star>
icon in the column Header to shrink Alerting Cockpit to display your favorite alerts only:
